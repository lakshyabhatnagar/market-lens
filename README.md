## About

MarketLens is a full-stack stock market tracking platform built with Next.js 15 that gives investors real-time market data, interactive TradingView charts, a personal watchlist, and AI-curated daily email digests — all in a sleek, dark-themed interface.

## Features

🏠 **Real-Time Market Dashboard**
Live market overview, stock heatmaps, top stories, and market quotes powered by TradingView widgets.

🔍 **Powerful Stock Search**
Search thousands of stocks via Finnhub API with instant results. Star stocks directly from search to add them to your watchlist.

⭐ **Personal Watchlist**
Track your favorite stocks with live mini charts, price data, and a scrolling ticker tape banner. Add or remove stocks with a single click.

📊 **In-Depth Stock Analysis**
Candlestick charts, baseline charts, technical analysis, company profiles, and financial statements for every stock.

📧 **AI-Powered Daily Email Summaries**
Personalized daily market digests generated by Google Gemini AI based on your watchlisted stocks. Delivered to your inbox with one-click unsubscribe.

🔔 **Smart Alerts & Notifications**
Welcome emails on sign-up and configurable stock alerts to keep you informed.

🔐 **Secure Authentication**
Full auth flow (sign-up, sign-in) via Better Auth with MongoDB adapter.

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- MongoDB Atlas account
- Finnhub API key
- Google Gemini API key
- Gmail App Password (for Nodemailer)

### Installation

```bash
# Clone the repository
git clone https://github.com/lakshyabhatnagar/market-lens.git
cd market-lens

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Fill in all required values (see below)

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

### Running Inngest (Background Jobs)

In a separate terminal:

```bash
npx inngest-cli@latest dev
```

## Environment Variables

Create a `.env.local` file in the root directory with:

```env
# MongoDB
MONGODB_URI=mongodb+srv://<user>:<password>@<cluster>.mongodb.net/<db>

# Better Auth
BETTER_AUTH_SECRET=your-secret-key
BETTER_AUTH_URL=http://localhost:3000

# Finnhub
FINNHUB_API_KEY=your-finnhub-api-key
NEXT_PUBLIC_FINNHUB_API_KEY=your-finnhub-api-key

# Nodemailer (Gmail)
NODEMAILER_EMAIL=your-email@gmail.com
NODEMAILER_PASSWORD=your-gmail-app-password

# Google Gemini AI
GEMINI_API_KEY=your-gemini-api-key

# App
NEXT_PUBLIC_BASE_URL=http://localhost:3000
```
